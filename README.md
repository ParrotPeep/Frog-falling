# Forg Falling

A Minecraft dropper-style browser game where you fall through tunnels and dodge obstacles.

## How to Play

- **Arrow Keys** or **A/D** to move left and right
- Dodge gray obstacles as you fall
- Reach the green finish line at the bottom to complete each level
- You respawn instantly when you hit an obstacle

## Features

- 8 hand-crafted levels with increasing difficulty
- Gravity-based falling physics
- Carousel level selector with difficulty ratings
- Instant respawn with death counter
- Progress bar showing how far you've fallen
- No external dependencies - runs entirely in a single HTML file

## Running Locally

Open `index.html` in any modern browser. No build step or server needed.
